# Инжинириум МГТУ — главная + страница курсов

Две страницы сайта на чистом HTML + CSS + vanilla JS.

## Что есть

- **Слайдер** на главной с автопроигрыванием, паузой по ховеру, точками и стрелками
- **Scroll reveal** — элементы появляются при скролле (4 типа анимации)
- **Счётчики** статистики с плавной анимацией цифр
- **Ступени обучения** — поэтапное появление
- **FAQ** — аккордеон
- **Tilt** — иконки наклоняются за курсором
- **Карточки курсов** — data-driven: все курсы лежат в массиве `coursesData` в `js/courses-data.js`. Добавить курс = дописать один объект в массив
- **Фильтр** курсов по возрасту и формату
- **Формы** — не перезагружают страницу, показывают «Отправлено ✓»
- **Шапка и подвал** — общие через `js/components.js`

## Как запустить

Открыть `index.html` или `courses.html` в браузере.

## Что можно докрутить

- Google Fonts с CDN — без инета шрифт не встанет. Можно скачать и подключить локально
- Изображения в `img/` — сжать png, весят многовато
- Формы — сейчас только имитация. Нужен бекенд или отправка на почту
- Нет адаптива под мобилки. Сейчас только desktop
- Favicon только 32×32. Для современных браузеров нужны ещё svg и webp-версии
- Вместо дублирования header/footer через JS можно сделать шаблонизатор (Pug, Nunjucks) при сборке
